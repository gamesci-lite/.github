## gamesci lite

写代码的时候假装在做游戏，做游戏的时候假装在写代码。

我们用 Rust 写后端，用 TypeScript 画前端，用 GraphQL 粘一切。  
如果 bug 修不好，就说它是 feature。

`cargo build --release` 是我们的战斗口号。

--- 

When writing code, pretend you're making a game.
When making a game, pretend you're writing code.

We write the backend in Rust,
Draw the frontend with TypeScript,
And glue everything together using GraphQL.

If a bug can't be fixed, call it a feature.

cargo build --release is our battle cry.
